# PROCommerce
O projeto PROCommerce tem como objetivo disponibilizar as principais funcionalidades de um e-commerce.

## Funcionalidades
- Autenticação
- Catálogo de Produtos
- Carrinho de Compras
- Sistema de Pagamentos
- Perfil de usuário


## Tecnologias
- .NET 8
- SQL Server
- RabbitMQ
- Microsserviços
- Arquitetura limpa
